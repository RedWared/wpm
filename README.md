# Make Wah-Package
A tool for build packages for Wah Package-Manager inspired on makepkg from arch

## Dependencies:
* bash
* dirname (coreutils)

> Installing dependencies on Arch Linux or derivatives
```
sudo pacman -S bash coreutils
```

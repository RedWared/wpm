# Make Wah-Package:

> Dependencies:
* bash
* basename
* cut
* sed

> Error codes:

| Exit code	| Cause						|
| ---		| ---						|
| -1		| Unkonow error.				|
| 0		| Normal exit.					|
| 1		| Main or configuration script not found.	|
| 2		| Configuration not found.			|
| 3		| Unknow option.				|
| 4		| Can't find WAHBUILD in actual directory.	|
| 5		| Important variable empity in WAHBUILD.	|
| 6		| File specified on "source" not found.		|
